# 제어구조



## 조건문

## 반복문

while

for

```js
var fruits=["apple", "banana", "graph"];
for(var i=0; i<fruits.length; i++){
    console.log(fruits[i]);
}
```



for-in : 키값을 순횐합니다.

```js
var obj={id:1, title:"hello"};
for(var k in obj) { // 키를 반환합니다.
    console.log(obj[k]);
}
```
